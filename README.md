# Todoo Hardware Management

A simple and lightweight Odoo module to manage IT hardware assets inside your company.

## 🚀 Features
- Track IT equipment (laptops, desktops, printers, etc.)
- Assign devices to employees
- Monitor device status (in use, available, broken)
- Keep inventory organized and controlled

## 📦 Installation
1. Copy the module to your Odoo addons directory
2. Restart Odoo
3. Activate developer mode
4. Install the module from Apps menu

## ⚙️ Requirements
- Odoo 19.0

## 🧩 Usage
- Go to Hardware menu
- Add devices
- Assign to users
- Track status and usage
